# Buchführung Klausurtrainer PWA

Diese Version funktioniert unabhängig von ChatGPT.

## Dateien
- index.html
- manifest.json
- service-worker.js
- icons/

## Lokal öffnen
Die Datei `index.html` kann direkt im Browser geöffnet werden.

## Als App nutzen
Am zuverlässigsten funktioniert die Installation, wenn du den Ordner online stellst, z. B. über GitHub Pages.

### GitHub Pages kurz
1. Neues GitHub Repository erstellen, z. B. `buchfuehrung-trainer`.
2. Alle Dateien aus diesem ZIP hochladen.
3. In GitHub: Settings → Pages → Deploy from branch → main → /root.
4. Die GitHub-Pages-URL auf dem Handy öffnen.
5. Im Browser-Menü: „Zum Startbildschirm hinzufügen“.

Der Lernstand wird lokal im Browser gespeichert.
